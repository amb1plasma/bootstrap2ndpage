AN ARTICLE ON USING COMPUTER SCIENCE TO IMPROVE ACCESSIBILITY

Computer science has been used to aid in accessibility in other ways, too. It has been used to create many different tools and methods to make the field as inclusive as possible. Examples include: eye tracking software to help paralysed people use computers; puff and blow switches; using alt tags on webpages so blind people can understand what is on the page; and braille keyboards so blind people can use computers.
